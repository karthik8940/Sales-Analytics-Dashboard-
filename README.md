# 📊 Sales Analytics Dashboard

## 📌 Project Overview

This project is an end-to-end Data Analytics solution built using Python, SQL, and Power BI. The objective is to analyze retail sales data, identify business trends, and provide actionable insights through an interactive dashboard.

The project covers the complete analytics workflow:

* Data Collection
* Data Cleaning
* Exploratory Data Analysis (EDA)
* SQL Analysis
* Data Visualization
* Dashboard Development
* Business Insights

---

## 🎯 Business Problem

Retail businesses generate large volumes of sales data. Analyzing this data helps organizations:

* Identify top-performing products
* Understand regional sales performance
* Monitor profit trends
* Track customer purchasing behavior
* Make data-driven business decisions

## 💡 Business Insights

Key findings from the analysis:

* Technology category generated the highest revenue.
* Certain regions consistently outperformed others.
* Sales increased significantly during peak seasons.
* A small percentage of customers contributed a large share of revenue.
* Some products generated high sales but low profit margins.
